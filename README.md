# Mouse Chaser

Android video game with SDL2 and C++

This app is a Video Game written in C++ and SDL2 library.
We use jni to bundle this app into Android.

Yes, we use native c++ code in an Android app.

The source can be found in:

    /app/jni/src/


## Libraries or Techs

    - SDL2
    - Make
    - JNI
    - C++
